The name of this project is KeenKeeper.

This is my 7th assignment of my web course “AI Powered Future Ready Web Dev (Batch-13)” by Programming Hero.
This website is based on friends management. There is a navbar with three buttons: Home, Timeline and Stats. In this website, some friends are shown as cards. Each card has image, name, how many days since contact, tags and status. When I click on a friend card, I can see full details of that friend. From there I can call, send message or make a video call.

In the Timeline section, all activity history like calls, messages and video calls are shown. I can also filter them by type.
In the Stats section, there is a pie chart which shows total number of calls, messages and video calls.

I have used HTML5, CSS3, Tailwind CSS, DaisyUI, JavaScript (ES6) and React JS to build this project.

Features of this website:

By clicking on a friend card, I can see full details of that friend.
I can filter activities like call, message and video call in the Timeline section.
In the Stats section, a pie chart shows all activity data (call, message, video call).
